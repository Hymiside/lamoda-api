# Тестовое задание Lamoda tech

В данном решении реализованы все методы, указанные в задании. Также дополнительно реализована логика работы с множеством складов. Идея такова, товар бронируется на ближайшем складе от места заказа. Рассчеты расстояний реализованы на уровне запросов в БД, с помощью PostGIS.
